AWS Scalable Architecture Study

Este estudo explora uma arquitetura cloud baseada em AWS projetada para suportar aplicações com milhares de usuários simultâneos.

Problema

Aplicação web com alto volume de acessos simultâneos necessitando escalabilidade e alta disponibilidade.

Arquitetura Proposta

  • Application Load Balancer
  • Auto Scaling
  • EC2 Containers
  • PostgreSQL
  • S3
